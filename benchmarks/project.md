# Calculator Project

## Goal

Build a simple calculator CLI app.

## Features

- **Add**: add two numbers (`add <a> <b>`)
- **Subtract**: subtract two numbers (`sub <a> <b>`)
- **Multiply**: multiply two numbers (`mul <a> <b>`)

## Out of Scope

- Division
- Floating point edge cases
- GUI / web interface

## Target

A working Python CLI. User runs `python calc.py add 3 5` and gets `8`.
