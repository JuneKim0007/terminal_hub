# /th:gh-implementation/implement — Implement a specific issue

Runs Steps 3–9 of the gh-implementation flow for a named or selected issue.

## Usage

The user can invoke this with:
- `/th:gh-implementation/implement` (will ask which issue)
- `/th:gh-implementation/implement #5` (direct slug)
- natural language: "implement issue 5" / "work on the auth bug"

Follow the full flow from `/th:gh-implementation` Steps 3–9.
Start from Step 3 if no issue is specified; jump to Step 4 if slug is known.
